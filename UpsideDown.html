Upside Down

Description

A platformer video game made by Joan Barduena and me using C++ (with SDL and pugi libraries). Both maps were made using Tiled.

The game consists on completing both levels and picking the maximum amount of coins while not dying. The main feature of this game is swapping gravity upside down to avoid enemies and move through the level.
