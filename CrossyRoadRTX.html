CrossyRoad RTX

Context

When I was a kid, I remember a period where every student played the same mobile game, Crossy Road. It was very popular because we were always competing to see who had the highest score of the class. This is why everytime I hear about Crossy Road it puts a smile on my face.

During my free time I wanted to challenge me with a little project of some weeks of work, so I decided to remake a mobile game that was simple and fun. Automatically I thought about CrossyRoad because it is a game that brings me back to the old good days.

Description

The idea of this project was to make CrossyRoad but with the current generation of graphics, this is why I chose Unreal Engine 5. Although Crossy Road was a mobile game, CrossyRoadRTX is not meant to be played in mobile or consoles, and I haven't released it since I don't want to infringe any third-party rights.

All the coding was made using blueprints because it is a very powerful visual scripting tool. All the assets were downloaded from Unreal Engine asset store except the charater that I modeled myself, the audios are the originals of the game and tried to recreate the original UI aswell.

There are still some bugs and features that I would like to add in a near future, but overall, I am very happy with the result.
